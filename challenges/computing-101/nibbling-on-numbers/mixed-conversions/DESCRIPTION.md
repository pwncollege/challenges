This is the capstone: every conversion you've learned, shuffled together.

A single byte can be written four different ways, and you've now practiced all of them:
- an **unsigned decimal** number, 0 to 255
- a **signed decimal** number, -128 to 127 (two's complement)
- two **hex** digits
- eight **binary** bits

`/challenge/mixed` shows you a byte in one of those forms and asks you to rewrite it in each of the others --- so a single value becomes all four forms, one after another.
Across the run you'll hit every pairing, including flipping a value between its signed and unsigned decimal readings, which are the same bits read two different ways.

Get every conversion right, one after another, to earn the flag.
Slip on one and it'll tell you exactly what went wrong --- then just run it again.
